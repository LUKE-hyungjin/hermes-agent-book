# Hermes Agent 한국어 가이드

Hermes Agent v0.6.0 기준 한국어 가이드 웹사이트.

## 개발

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 배포

Cloudflare Pages에 연결하여 자동 배포.

## 구조

- `content/` — Markdown 챕터 파일
- `src/` — 프론트엔드 소스
- `scripts/` — 빌드 스크립트
- `PLAN.md` — 프로젝트 계획서
- `REQUIREMENTS.md` — 요구사항 정리
