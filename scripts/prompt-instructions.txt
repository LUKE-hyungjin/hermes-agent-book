Hermes Agent v0.5.0 한국어 가이드 작성 지침:

1. 모든 내용은 한국어로 작성
2. 톤: 친절하지만 전문적인 한국어 (존댓말 사용)
3. 각 챕터 구조:
   - 첫 줄: # 번호. 제목
   - 인용구로 한 줄 요약 (> ...)
   - ## 섹션 제목으로 세부 내용 구조화
   - 코드 블록은 ```bash, ```python, ```yaml 등 언어 명시
   - 표(table) 적극 활용
   - :::tip, :::warning, :::info 박스 사용 가능
   - 마지막에 이전/다음 링크

4. 각 챕터당 2000-4000자 분량
5. 초보자도 이해할 수 있게 설명
6. 실제 사용 예시(코드) 포함
7. 공식 문서 내용을 충실히 반영하되, 한국 독자에게 맞게 재구성
